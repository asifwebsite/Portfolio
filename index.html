Animals Adoption Website

Adopt-Me

In this project, I built a website using React. The site connects to an API that stores a list of animals and displays them. Users can select a location, animal, and breed in a form that filters the animal list, displaying only relevant entries. By clicking on an animal, the user is redirected to a detailed page, with a description, images, and an option to adopt. I built this website using the latest features in React, including hooks, effects, context, and portals, with tools like Parcel, ESLint, Prettier, and React Router.

AI-Driven Image Creation

ImageGenerationOpenAI

ImageGenerationOpenAI is a web application that leverages the power of OpenAI to generate images based on textual prompts. This project demonstrates the capabilities of OpenAI's language model in generating visual content from natural language descriptions.
How It Works:
Users enter a textual prompt describing the image they want to generate. The application sends this prompt to OpenAI's GPT-3.5 model, which interprets the text and generates a corresponding image. The generated image is then displayed to the user.
